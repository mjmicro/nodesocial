# Gamification System

## XP & Levels

Every action earns XP. Levels unlock abilities.

| Level | Title | Unlocks |
|-------|-------|---------|
| 1–2 | Observer | Report issues, join missions, verify |
| 3–4 | Contributor | Create missions, propose solutions |
| 5–7 | Changemaker | Coordinate missions, mentor others |
| 8–10 | Problem Solver | City-scope missions, expert voting |
| 11+ | Catalyst | National missions, community moderator |

## Daily XP Cap

**200 XP/day maximum.** Prevents grinding. Quality over quantity.

## Daily Quests

- 4 quick tasks per day (~2 min each)
- Contextual to neighborhood activity (not generic)
- Completing all 4 = bonus XP multiplier
- Quests rotate daily at midnight local time

## Badges

Earned through specific verified contributions only — never for passive activity.

| Badge | Earned by |
|-------|-----------|
| First Responder | First to report an issue that gets confirmed |
| Fixer | Contributed to 5+ resolved missions |
| Truth Seeker | 10+ accurate disputes accepted |
| Neighborhood Hero | Top contributor in neighborhood for a month |
| Bridge Builder | Connected two separate neighborhood missions |
| Streak Master | Maintained a 30-day streak |
| City Catalyst | Triggered a city-scope escalation |
| Global Solver | Contributed to a global-scope mission |

## Streaks

- Daily activity streak (any verified action counts)
- **2 free streak freezes per month**
- Streak bonus (additive on top of base XP):

| Day range | Bonus |
|-----------|-------|
| Day 1–7 | +5 XP/day |
| Day 8–30 | +10 XP/day |
| Day 31–90 | +15 XP/day |
| Day 91+ | +20 XP/day |

## Leaderboards

- **Neighborhood leaderboard** — weekly reset, available from Level 1
- **City leaderboard** — unlocks at Level 8
- Ranked by **verified impact**, not raw XP
- Always display "X XP to next rank" — never show a dead end

## Anti-Addiction Rules

- Daily XP cap: 200/day
- "You're caught up" endpoint — app explicitly tells users when there's nothing new
- No notifications between **10PM–7AM** (user's local time, default, adjustable)
- Weekly impact summary replaces daily stats
